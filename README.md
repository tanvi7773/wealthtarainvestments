# Wealth Tara Investments

A professional investment portfolio and services website showcasing investment opportunities and expertise.

## 📋 Overview

Wealth Tara Investments is a responsive web application designed to present investment services, portfolio information, and facilitate client communication through an intuitive interface.

```
wealthtarainvestments/
├── index.html           # Main landing page
```

## 🎨 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Professional Styling** - Modern gradient design with smooth transitions
- **Interactive Elements** - Hover effects, smooth scrolling, and form validation
- **Contact Form** - Client inquiry submission with validation
- **Portfolio Display** - Showcase of investment opportunities with card layouts
- **Cross-browser Compatible** - Works on Chrome, Firefox, Safari, and Edge

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- GitHub account (for hosting)

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/tanvi7773/wealthtarainvestments.git
cd wealthtarainvestments
```

2. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# Or simply double-click the file
```

## 📱 GitHub Pages Setup

To host your website online using GitHub Pages:

1. Go to your repository settings: https://github.com/tanvi7773/wealthtarainvestments/settings
2. Navigate to **Pages** section (on the left sidebar)
3. Under "Source", select the branch as **main** and folder as **/ (root)**
4. Click **Save**
5. Your site will be live at: `https://tanvi7773.github.io/wealthtarainvestments/`

## 🔧 Customization

### Update Colors
Open `styles.css` and modify the gradient colors:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Update Navigation Links
Edit navigation in your HTML files:
```html
<nav>
  <a href="index.html">Home</a>
  <a href="investments.html">Investments</a>
  <a href="portfolio.html">Portfolio</a>
  <a href="contact.html">Contact</a>
</nav>
```

### Add More Sections
Create new HTML files and link them in the navigation menu.
